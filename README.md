# 🎮 Python Mini Games Collection

A simple Python project that includes two classic games:

* 🎲 Dice Roll Simulator
* ✊✋✌️ Rock Paper Scissors

Built using Python's built-in `random` module, this project is perfect for beginners learning basic programming concepts like conditionals, loops, and user input.

---

## 📌 Features

* Random dice roll generator (1–6)
* Interactive Rock Paper Scissors game
* User vs Computer gameplay
* Simple and clean command-line interface
* Beginner-friendly code structure

---

## 🛠️ Technologies Used

* Python 3
* `random` module (for generating randomness)

---

## 🚀 How to Run

1. Make sure Python is installed on your system
2. Clone this repository or download the `.py` file
3. Open terminal / command prompt
4. Navigate to the project folder
5. Run the script:

```bash
python game.py
```

---

## 🎲 Dice Roll Game

* Simulates rolling a six-sided dice
* Generates a random number between 1 and 6
* Can be extended to roll multiple dice

**Example Output:**

```
You rolled: 4
```

---

## ✊ Rock Paper Scissors

* Play against the computer
* Choose between:

  * Rock
  * Paper
  * Scissors
* Computer randomly selects its move
* Winner is decided based on standard rules

**Rules:**

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock

**Example Gameplay:**

```
Enter your choice (rock/paper/scissors): rock
Computer chose: scissors
You win!
```
